# 📊 HR Analytics Dashboard

This project is an interactive **HR Analytics Dashboard built using Power BI** to analyze employee data and uncover key insights related to workforce attrition, demographics, and job satisfaction.

---

## 🚀 Project Overview

The dashboard analyzes HR data of **1,470 employees** to identify key patterns in employee attrition and workforce distribution across departments, age groups, education fields, and job roles.

It helps HR teams make **data-driven decisions** to improve employee retention and workforce planning.

---

## 🛠️ Tools & Technologies Used

- Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Modeling & KPI Calculations)  
- Microsoft Excel  

---

## 📌 Key Insights

- Overall attrition rate: **16.12% (237 employees)**
- Highest attrition observed in the **R&D department (56%+ of total exits)**
- Majority attrition found in the **25–34 age group**
- Workforce distribution analyzed across gender, education, and job roles
- Job satisfaction analysis revealed variation across different roles

---

## 📈 Dashboard Features

- Interactive KPI cards (Attrition, Headcount, Active Employees, Average Age)
- Department-wise attrition analysis
- Age, gender, and education-based segmentation
- Job role and satisfaction analysis
- Drill-down and filter-enabled visuals for deep insights

---

## 🎯 Objective

To identify key drivers of employee attrition and provide actionable insights that support **HR decision-making and retention strategies**.
